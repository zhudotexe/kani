Sure! Here is a simple Python function to calculate the factorial of a given number:
```
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)
```
You can use this function by calling it with a number, like this:
```
print(factorial(5)) # prints 120
```
